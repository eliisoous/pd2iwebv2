Quelles données personnelles nous collectons et pourquoi

Formulaires de contact

Lorsque vous nous contactez via nos formulaires de contact, nous collectons :

  • • Votre nom et adresse e-mail
  • • Le contenu de votre message
  • • Adresse IP et horodatage à des fins de sécurité

Objectif : Répondre à vos demandes et fournir un support client.

Téléchargements de médias

Si vous téléchargez des images sur le site web, vous devriez éviter de télécharger des images contenant des données de localisation intégrées (EXIF GPS). Les visiteurs du site web peuvent télécharger et extraire toutes les données de localisation des images présentes sur le site.

⚠️ Veuillez supprimer les données de localisation des images avant de les télécharger pour protéger votre vie privée.

Cookies

Nous utilisons des cookies pour améliorer votre expérience de navigation. Si vous laissez un commentaire ou nous contactez, vous pouvez choisir d'enregistrer votre nom et votre adresse e-mail dans des cookies pour votre commodité. Ces cookies dureront un an.

Pour des informations détaillées sur notre utilisation des cookies, veuillez consulter notre Politique relative aux cookies.

Analyses du site web

Nous collectons des données d'analyse de base du site web pour comprendre comment les visiteurs utilisent notre site :

  • • Pages visitées et temps passé sur le site
  • • Type de navigateur et informations sur l'appareil
  • • Localisation géographique générale (niveau pays/ville)

Objectif : Améliorer les performances de notre site web et l'expérience utilisateur.

Contenu intégré provenant d'autres sites web

Les articles de ce site peuvent inclure du contenu intégré (par exemple, vidéos, images, articles, etc.). Le contenu intégré provenant d'autres sites web se comporte exactement de la même manière que si le visiteur s'était rendu sur l'autre site web.

Ces sites web peuvent collecter des données vous concernant, utiliser des cookies, intégrer un suivi tiers supplémentaire et surveiller votre interaction avec ce contenu intégré, y compris le suivi de votre interaction avec le contenu intégré si vous avez un compte et êtes connecté à ce site web.

🔴 Important : Nous ne pouvons pas contrôler les pratiques de confidentialité du contenu tiers intégré. Veuillez consulter les politiques de confidentialité de ces services externes.

Vos droits en vertu du RGPD

Vous avez le droit de :

  • 📄
    Accès - Demander une copie de vos données personnelles
  • ✏️
    Rectification - Corriger des données inexactes
  • 🗑️
    Effacement - Demander la suppression de vos données
  • ⏸️
    Limitation - Limiter la manière dont nous traitons vos données
  • 📦
    Portabilité - Recevoir vos données dans un format structuré
  • 🚫
    Opposition - Refuser certains traitements de données

Comment exercer vos droits :

Contactez-nous à admin@pd2i.com avec votre demande. Nous répondrons dans un délai de 30 jours.

✅ Toutes les demandes sont traitées gratuitement sauf si elles sont excessives ou répétitives.
